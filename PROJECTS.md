## WORKFLOW FÜR NEUES PROJEKT

1. **GitHub** → github.com/treffenix-dev → New Repository → Name: `[kunde]-website`
2. **Claude Code** → code.claude.ai → New Session → Repo verbinden
3. **Vercel** → vercel.com → Add New Project → Repo importieren
4. **Hier eintragen** → PROJECTS.md updaten
5. **CLAUDE.md im neuen Repo** → Projektdaten eintragen (Kunde, Farben, Fonts, Daten)
